# 🔵 Blue Notes

> Aplicativo de notas minimalista com tema azul, construído em C# com .NET MAUI.

## 📦 Versão
`v0.0.0.0.1` — Release inicial (WIP)

## ✨ Funcionalidades
- Criação e edição de notas com rich text
- Cadernos e etiquetas
- Busca global em tempo real
- Arquivo, lixeira e recuperação
- Notas fixadas com drag & drop
- Proteção por PIN/Biometria
- Exportação em .txt e .pdf
- Sidebar animada
- Tema azul dark/light
- Configurações e preferências

## 🛠️ Stack
- .NET MAUI (C#)
- SQLite local
- MVVM Architecture
